Setup requirements

1. install java jdk1.8
2. install eclipse
3. create java project with selenium Java Client libraries
4. download Geckodriver
5. download chromedriver
6. download selenium-server-standalone-3.13.0

Execution :  launch cmd  prompt
	     run  java -jar selenium-server-standalone-3.13.0 -role hub (notedown the server address in my case it is http://10.0.0.64:4444/wd/hub)
	     
	     now you can run the selenium script
